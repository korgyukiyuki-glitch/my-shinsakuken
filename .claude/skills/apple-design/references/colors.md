# カラーパレット

## 基本方針
- 色数は最小限（メイン1色 + グレースケール + アクセント1色）
- 背景は白 or 極薄グレー（`#f5f5f7`）
- テキストは真っ黒を避け、ダークグレー（`#1d1d1f`）を使用
- カラフルにしすぎない。抑制が洗練を生む

## Apple風カラースケール

### テキスト
| 用途 | カラーコード | 説明 |
|------|-------------|------|
| 見出し | `#1d1d1f` | ほぼ黒、やや柔らかい |
| 本文 | `#424245` | ダークグレー |
| サブテキスト | `#6e6e73` | ミディアムグレー |
| キャプション | `#86868b` | ライトグレー |

### 背景
| 用途 | カラーコード | 説明 |
|------|-------------|------|
| メイン背景 | `#ffffff` | 白 |
| セクション背景（交互） | `#f5f5f7` | Apple標準の薄グレー |
| ダークセクション | `#1d1d1f` | ほぼ黒 |
| カード背景 | `#ffffff` | 白 |

### アクセント
| 用途 | カラーコード | 説明 |
|------|-------------|------|
| プライマリリンク | `#0071e3` | Apple Blue |
| プライマリリンク hover | `#0077ed` | やや明るく |
| 成功 | `#34c759` | Green |
| 注意 | `#ff9500` | Orange |

### 禁止
- `#7c3aed` 系の紫グラデーション
- レインボーグラデーション
- ネオンカラー
- 3色以上のグラデーション

## CSS変数テンプレート
```css
:root {
  --color-text-primary: #1d1d1f;
  --color-text-secondary: #424245;
  --color-text-tertiary: #6e6e73;
  --color-text-caption: #86868b;
  --color-bg-primary: #ffffff;
  --color-bg-secondary: #f5f5f7;
  --color-bg-dark: #1d1d1f;
  --color-accent: #0071e3;
  --color-accent-hover: #0077ed;
}
```
