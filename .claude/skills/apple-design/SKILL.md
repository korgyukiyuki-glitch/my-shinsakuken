# Apple Design Skill

## Overview
Apple風のWebデザインを生成するためのスキル。
Apple.comのデザイン言語に基づき、洗練されたミニマルなUIを構築する。

## When to Use
- ランディングページ、プロダクトページ、マーケティングサイトの作成・リデザイン時
- 「Apple風」「洗練された」「ミニマル」「モダン」なデザインを求められた時

## Design Rules

### 必ず守ること
1. **余白を大胆に取る** — セクション間は最低 `100px`、要素間は `24px` 以上
2. **1カラムレイアウト** — メインコンテンツは中央揃え、最大幅 `980px`
3. **フォントサイズは大きく** — ヒーロー見出しは `48px`〜`64px`、本文は `17px`〜`21px`
4. **色数を絞る** — メイン1色 + グレースケール + アクセント1色
5. **グラデーションは控えめ** — 使う場合はモノクロ系 or 同系色の微妙なグラデーション
6. **角丸は統一** — `12px`〜`20px` で統一（小要素は `8px`）
7. **影は薄く繊細に** — `box-shadow: 0 4px 24px rgba(0,0,0,0.08)`

### 絶対にやらないこと
- 紫系のグラデーション背景
- 過度なアニメーション
- 3色以上のカラフルな配色
- 詰め込みすぎたレイアウト
- 太い枠線（border）
- テキストの影（text-shadow）
