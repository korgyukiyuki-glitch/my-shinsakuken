ACTUAL APP SCREENS

01 — EXAMINATIONS

4つの検査を、この1本で。

口腔機能の評価から歯周検査まで。診療でよく使う検査をカバーします。

ADULT / SENIOR — 成人・高齢者

口腔機能低下症

K1〜K7の7項目検査で口腔機能低下を評価・判定します。

  • TCI舌苔チャート（9ゾーン）
  • オーラルディアドコキネシス タイマー内蔵（pa/ta/ka）
  • EAT-10・RSST対応

PEDIATRIC — 18歳未満

口腔機能発達不全症

C1〜C17の17項目チェックリストで評価。問題点と管理計画の整理までつながります。

  • 咀嚼・嚥下・構音の機能評価
  • 口唇閉鎖力・舌圧スクリーニング
  • 管理計画書の自動生成

MASSETER NEW

咬筋検査モード

顎関節症・咀嚼筋痛障害の診療で、咬筋活動の測定値を記録できます。

  • 測定値は手入力（測定機器を選びません）
  • 結果報告書PDFを出力
  • 検査記録を患者ごとに保存

PERIODONTAL — 歯周検査

歯周検査

歯周精密検査・歯周基本検査を、歯式チャートでそのまま記録できます。

  • 精密6点法・4点法・基本1点法
  • BOP・排膿・動揺度・PCRの記録
  • 平均PD・BOP率・PCR率・PISAを自動集計

※ 検査結果の記録・PDF出力用の機能です。レセコン連携には対応していません。

02 — FEATURES

現場で、すぐ使える。

検査を入力すれば、判定も書類もアプリが引き受けます。

検査・自動判定

TCI・舌圧・オーラルディアドコキネシス（内蔵タイマーで計測）などの検査結果を入力すると、該当/非該当を自動判定。

レーダーチャート

7項目の検査結果をレーダーチャートで可視化。患者説明にそのまま使えます。

管理計画書・紹介状

検査結果から管理計画書をワンタップでPDF出力。検査結果を含む紹介状もテンプレートから作成できます。

算定ガイド

診断レベルに応じた保険算定項目と点数をアプリ内で自動表示。算定漏れを防ぎます。

03 — SECURITY

患者データを、安全に。

医療情報を扱うアプリとして、セキュリティを最優先に設計しています。

PINロック

アプリ起動時にPIN認証。第三者のアクセスを防ぎます。

暗号化保存

患者データはデバイス内に暗号化して保存します。

暗号化バックアップ

AES-256-GCMで暗号化してクラウドへ。端末間の同期も安全に行えます。
