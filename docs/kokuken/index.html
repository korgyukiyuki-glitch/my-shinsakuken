2つのモード。

小児から高齢者まで、口腔機能管理をカバー。

👶
小児（18歳未満）

口腔機能発達不全症

C1〜C17の17項目チェックリストで評価。歯式チャート対応。

  • 咀嚼・嚥下・構音の機能評価
  • 口唇閉鎖力・舌圧スクリーニング
  • 管理計画書の自動生成
🧓
成人・高齢者

口腔機能低下症

K1〜K7の7項目検査で口腔機能低下を評価・判定。

  • TCI舌苔チャート（9ゾーン）
  • ODKタイマー内蔵（pa/ta/ka）
  • EAT-10・RSST対応
