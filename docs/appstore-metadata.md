## 説明文

複数の医院の診察券を、スマホ1つで管理できるアプリです。

歯科、内科、眼科、皮膚科、整形外科――
通っている全ての医院の診察券をまとめて登録。
受付ではスマホを見せるだけ。もう診察券を忘れる心配はありません。

【主な機能】
- 診察券の登録・管理（枚数無制限・全診療科対応）
- 患者番号の大画面表示（受付で見せるだけ）
- QRコード表示
- 予約メモ＆プッシュ通知リマインダー
- 診療履歴の記録
- お薬情報の管理
- 生体認証（Face ID / Touch ID）ロック

【安心のプライバシー設計】
すべてのデータはお使いの端末内にのみ保存。
サーバーへの送信やクラウド保存は一切行いません。
広告・トラッキングもありません。

【医院への導入不要】
このアプリは患者さんが自分で使えるアプリです。
医院側にシステム導入を求めることはありません。
医院名と患者番号を手入力するだけで、すぐに使い始められます。

【完全無料】
すべての機能を無料でご利用いただけます。
診察券の登録枚数に制限はありません。

## 審査メモ（Review Notes）
This app allows patients to manage their medical appointment cards digitally.
All data is stored locally on the device only - no server communication.
No account creation is required. Users can register clinic information manually.
The app does not provide medical advice or diagnosis.

主な操作手順:
1. アプリ起動 → ウェルカム画面でスワイプ → プロフィール設定
2. ホーム画面 → 「医院を追加」→ 医院名・患者番号を入力
3. 診察券一覧で登録した診察券をタップ → 患者番号・QRコード表示
4. 予約メモ追加 → 日時・医院・内容を入力 → リマインダー通知
