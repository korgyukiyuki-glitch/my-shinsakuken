プライバシーポリシー

Medical Craft Laboratory（以下「当社」）は、マイ診察券アプリ（以下「本アプリ」）におけるお客様の個人情報の取り扱いについて、以下のとおりプライバシーポリシーを定めます。

1. 収集する情報

本アプリは、以下の情報をお客様の端末内にのみ保存します。サーバーへの送信は一切行いません。

  • お名前（プロフィール設定で任意入力）
  • 医院名、患者番号（診察券登録情報）
  • 予約日時、診療内容などのメモ
  • 診療履歴の記録
  • お薬情報の記録

2. データの保存場所

すべてのデータはお客様のスマートフォン端末内にのみ保存されます。外部サーバーへの送信、クラウドへのアップロードは行っておりません。端末を変更された場合、データの引き継ぎはできません。

3. 第三者への提供

当社は、お客様の個人情報を第三者に提供、販売、共有することはありません。

4. アクセス権限

本アプリは以下の端末機能へのアクセスを求める場合があります。いずれも任意であり、許可しなくても基本機能はご利用いただけます。

  • カメラ：お薬手帳や診察券の撮影（将来機能）
  • 通知：予約リマインダーのプッシュ通知
  • 生体認証（Face ID / Touch ID）：アプリロック機能

5. 分析・トラッキング

本アプリは、利用状況の分析ツールや広告トラッキングツールを使用していません。お客様の行動を追跡することはありません。

6. 子どものプライバシー

本アプリは13歳未満のお子様を対象としていませんが、保護者の方がお子様の診察券を管理する目的でご利用いただくことは可能です。

7. セキュリティ

本アプリは端末内のセキュアストレージを使用し、生体認証によるアプリロック機能を提供しています。ただし、端末自体のセキュリティはお客様の責任となります。

8. プライバシーポリシーの変更

当社は、必要に応じて本ポリシーを変更することがあります。変更があった場合は、本ページにて更新いたします。

9. お問い合わせ

プライバシーに関するお問い合わせは、以下のメールアドレスまでご連絡ください。

メールアドレス: support@medicalcraft-lab.com
