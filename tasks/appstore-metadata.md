## 説明文 (4000文字以内)

```
診察券を、スマホひとつで。

歯科、内科、眼科、皮膚科、整形外科 — 通っている医院の診察券をすべてアプリで管理できます。もう財布の中から診察券を探す必要はありません。

■ 主な機能

【診察券のデジタル管理】
・医院名、患者番号、診療科を登録
・カラフルなカードデザインで一目で識別
・枚数無制限、すべて無料

【受付をスムーズに】
・QRコードを画面に表示して、受付でスキャン
・QR非対応の医院でも、患者番号を大画面表示
・どの医院でもすぐに使えます

【予約メモ & リマインダー】
・次回の予約をアプリに記録
・プッシュ通知で予約日をリマインド
・もう予約を忘れません

【診療履歴の記録】
・受診日、治療内容、担当医をメモ
・通院の記録を時系列で管理

【お薬手帳】
・処方されたお薬を記録
・薬品名、用量、用法を管理
・複数の医院の処方をまとめて確認

■ プライバシー & セキュリティ

・データはすべて端末内に保存（サーバーに送信しません）
・広告なし、トラッキングなし
・Face ID / Touch IDでアプリをロック

■ こんな方におすすめ

・複数の医院に通っている方
・診察券をよく忘れる方
・通院の記録を整理したい方
・お薬の管理をまとめたい方

医院側のシステム導入は不要。今すぐ、ひとりで始められます。

© 2026 Medical Craft Laboratory
```

## 審査メモ (App Review Information)

```
このアプリは患者さん個人が自分の診察券情報を管理するためのアプリです。
医療判断機能は含まれておりません。

テスト手順：
1. アプリを起動し、プロフィール（氏名）を入力
2. ホーム画面 →「医院を追加」をタップ
3. 医院名（例：「テスト歯科」）と患者番号（例：「12345」）を入力して保存
4. 登録した診察券をタップ → QRコード・患者番号の表示を確認
5. 予約メモを追加 → リマインダー通知設定を確認

デモアカウント：不要（ローカルデータのみ、サーバー通信なし）
```
